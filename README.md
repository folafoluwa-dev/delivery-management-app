# Delivery Management System

A comprehensive logistics and delivery management web application designed to streamline order placement, tracking, and delivery operations.

## ✨ Key Features
- Customer order placement and real-time tracking
- Driver assignment and management system
- Live delivery status updates
- Admin dashboard for monitoring all orders
- User roles (Customer, Driver, Admin)
- Interactive order history
- Notification system (in-app)

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS, Leaflet.js (for maps - planned)
- **Backend:** Django, Django REST Framework
- **Real-time Updates:** Django Channels + WebSockets
- **Database:** PostgreSQL
- **Containerization:** Docker, Docker Compose
- **Authentication:** JWT
